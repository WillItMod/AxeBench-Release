# AxeBench-Release

Release-facing documentation and checksums for **AxeBench v3.0.0**.

## Downloads

Download artifacts from GitHub Releases:

- Latest stable: `https://github.com/WillItMod/AxeBench-Release/releases/latest`
- All releases: `https://github.com/WillItMod/AxeBench-Release/releases`

Artifacts:

- Windows: `AxeBench_v3.0.0.exe`
- Linux: `AxeBench_v3.0.0_linux.bin`
- Android client: `axebench_client_v1.2.apk`

Checksums:

- Stable: `checksums/v3.0.0.sha256`
- RC1e (legacy): `checksums/RC1e.sha256`
